# Статический сайт на HTML5 с Bootstrap 5

Простой и современный статический сайт, созданный с использованием HTML5 и фреймворка Bootstrap 5.

## Структура проекта

```
static-site/
├── index.html          # Главная страница сайта
├── css/
│   └── style.css       # Пользовательские стили
├── js/
│   └── main.js         # Пользовательские скрипты
├── vendor/             # Локальные копии библиотек (Bootstrap)
│   ├── css/
│   │   └── bootstrap.min.css
│   └── js/
│       └── bootstrap.bundle.min.js
└── images/             # Папка для изображений
```

## Особенности

- **Адаптивный дизайн** - сайт корректно отображается на всех устройствах
- **Bootstrap 5.3.2** - используется локальная копия фреймворка (файлы находятся в папке `vendor/`)
- **Автономная работа** - не требует подключения к интернету для загрузки библиотек
- **Современные компоненты**:
  - Навигационная панель с мобильным меню
  - Hero-секция с призывом к действию
  - Секция "О нас" с изображением
  - Карточки услуг с анимацией при наведении
  - Форма обратной связи
  - Подвал (footer)
- **Плавная прокрутка** к секциям
- **Чистый код** с комментариями на русском языке

## Как использовать

1. Откройте файл `index.html` в любом современном браузере
2. Для редактирования откройте проект в вашем любимом редакторе кода
3. Добавьте свои изображения в папку `images/`
4. Настройте содержимое под ваши нужды

## Требования

- Современный браузер (Chrome, Firefox, Safari, Edge)
- Интернет не требуется (все библиотеки находятся локально в проекте)
